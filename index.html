Pembayaran dilakukan di luar platform (via admin). Pengembalian dana dipertimbangkan secara kasus per kasus dalam 7 hari sejak pembelian jika kode akses belum pernah digunakan.

Privacy Usage

1. Data yang Kami Kumpulkan
NotaOTO LITE menyimpan data berikut di perangkat Anda sendiri (localStorage browser):
• Nama toko dan nomor WhatsApp toko
• Nama pelanggan dan daftar belanja
• Foto QRIS pembayaran (jika diunggah)
• Kode akses premium & masa berlaku
• Preferensi tema dan layout nota

2. Kami Tidak Mengumpulkan
• Data pribadi pengguna yang dapat diidentifikasi (nama, alamat, email)
• Riwayat transaksi penjualan
• Lokasi geografis
• Data kontak pelanggan dari buku alamat
• Informasi pembayaran atau kartu kredit
• Cookie pelacakan iklan

3. Penyimpanan Data
Seluruh data nota disimpan secara lokal di browser Anda menggunakan localStorage. Tidak ada data transaksi yang dikirim, disimpan, atau diproses di server kami. Data akan hilang jika Anda menghapus cache browser atau menggunakan perangkat/ganti browser lain.

4. Data Verifikasi
Saat aktivasi kode premium, kode akses dikirim ke server verifikasi (Google Apps Script) untuk validasi. Server hanya menerima kode akses - tidak ada data toko, pelanggan, atau barang yang dikirimkan.

5. Layanan Pihak Ketiga
Aplikasi ini menggunakan layanan pihak ketiga:
• Google Apps Script - verifikasi kode akses
• WhatsApp Web - pengiriman nota (atas tindakan pengguna)
• Google Fonts & CDN jsDelivr - muat font & library
• Lucide Icons - ikon SVG
Masing-masing pihak ketiga memiliki kebijakan privasi sendiri.

6. Keamanan
Karena data disimpan di perangkat Anda, keamanan data bergantung pada keamanan perangkat dan browser Anda. Kami merekomendasikan untuk tidak menggunakan aplikasi ini di perangkat publik/shared.

7. Hak Anda
Anda dapat menghapus seluruh data kapan saja dengan membersihkan localStorage browser (Settings → Privacy → Clear Data) atau melalui fitur Logout yang menghapus sesi premium.

8. Perubahan Kebijakan
Setiap perubahan pada kebijakan privasi akan diumumkan di halaman ini. Dengan terus menggunakan layanan setelah perubahan, Anda menyetujui kebijakan yang diperbarui.

9. Kontak
Untuk pertanyaan lebih lanjut, hubungi admin melalui WhatsApp yang tersedia di halaman aktivasi.
